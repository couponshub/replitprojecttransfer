Build a full-stack marketplace web application called “CouponsHub X”.

Tech Stack:
- Backend: Node.js + Express
- Database: PostgreSQL
- Auth: JWT-based authentication
- Password hashing: bcrypt
- Frontend: EJS or React (server-side rendering preferred)
- Clean scalable folder structure (MVC pattern)

==================================================
DATABASE STRUCTURE
==================================================

Use PostgreSQL tables:

1. users
- id (PK)
- name
- email (unique)
- password (hashed)
- role (admin | user)
- created_at

2. categories
- id (PK)
- name
- image
- created_at

3. shops
- id (PK)
- name
- description
- banner_image
- logo
- address
- whatsapp_number
- category_id (FK)
- is_premium (boolean default false)
- commission_percentage
- subscription_active (boolean)
- featured (boolean)
- created_at

4. products
- id (PK)
- shop_id (FK)
- name
- description
- price
- image
- created_at

5. coupons
- id (PK)
- shop_id (FK)
- code
- type (percentage | flat | free_item | flash)
- value
- is_active
- expiry_date
- created_at

6. orders
- id (PK)
- user_id (FK)
- total_amount
- discount_amount
- final_amount
- status (pending | confirmed | completed)
- created_at

7. order_items
- id (PK)
- order_id (FK)
- product_id (FK)
- quantity
- price

==================================================
AUTHENTICATION SYSTEM
==================================================

1. Create two login pages:
   - /admin-login
   - /login

2. Use users table for authentication.
3. Hash passwords using bcrypt.
4. Verify securely.
5. Generate JWT token.
6. Store token in httpOnly cookie.

After login:
- If role = admin → redirect to /admin-dashboard
- If role = user → redirect to /home
